Welcome to the **Advanced Java Shopping Web Application**! This project is a fully functional e-commerce platform built using **Java Servlet** and **JSP**, designed to provide a seamless shopping experience. The application includes features like product searching, detailed product views, a shopping cart, and an admin panel for efficient management.

## üìã Features

### User Features:
- **Homepage**: A welcoming and user-friendly landing page.
- **Login & Signup**: Secure authentication for users to access their accounts.
- **Product Listing**: Browse through a variety of products with pagination support.
- **Search Functionality**: Quickly search for products by title, description, or price.
- **Product Details**: View detailed descriptions and images of each product.
- **Add to Cart**: Seamlessly add products to your cart and review them before purchase.

### Admin Features:
- **Admin Panel**: Manage users and products with options to:
  - View, add, and edit products.
  - Manage user accounts efficiently.

## üöÄ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Java Development Kit (JDK)**: Version 8 or higher.
- **Apache Tomcat Server**: Version 9.0 or higher.
- **MySQL Database**: For storing user and product data.

### Installation Steps

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/Advanced_java_shopping.git
    cd advanced-java-shopping
    ```

2. **Set up the Database**:
   - Create a MySQL database named `shopping_db`.
   - Import the provided `shopping_db.sql` file to set up the database schema and populate it with sample data.

3. **Configure Database Credentials**:
   - Update the `db.properties` file with your MySQL database username and password.

4. **Deploy the Application**:
   - Build the project and generate the `shopping.war` file.
   - Copy the `shopping.war` file to the `webapps` directory of your Tomcat server.
   - Start the Tomcat server and navigate to `http://localhost:8080/shopping`.
